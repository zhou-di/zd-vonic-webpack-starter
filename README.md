# How to Use

- quick start

``` bash
git clone https://github.com/wangdahoo/vonic-webpack-starter.git
cd vonic-webpack-starter
npm install -g yarn
yarn
```

- development

```bash
npm run dev
```

- build

```bash
npm run build
```
